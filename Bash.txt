### Next Steps for the Platform
1. **GitHub Action:** Create an action in your monorepo so that when someone pushes a folder with `manifest.json` and `game.wasm`, it automatically updates `registry.json`.
2. **CDN Hosting:** Host `registry.json` and the `.wasm` files on GitHub Pages (it's free and can act as your CDN). Update `DefaultRegistryURL` in `main.go` to point to it.
3. **Graphics:** WASM perfectly handles TTY escape sequences. A hacker can write a game using libraries like `crossterm` in Rust, compile to WASI, and it will render full-color TUI graphics inside your launcher!
